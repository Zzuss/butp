# 华为云PDF服务部署指南

## 方案概述

将PDF服务部署到华为云服务器，提供公网HTTPS访问，完美解决网络连通性和安全问题。

## 架构图

```
浏览器 → https://butp.tech → https://pdf.你的域名.com → PDF服务
```

## 部署步骤

### 1. 准备华为云服务器

**推荐配置：**
- 实例规格：`s6.large.2`（2核4GB）或更高
- 镜像：Ubuntu 22.04 LTS
- 带宽：5Mbps+
- 安全组：开放22, 80, 443, 8443端口

### 2. 域名和SSL配置

**选项A：使用免费域名**
```bash
# 申请免费域名（如：freenom.com）
# 然后在华为云DNS解析中添加A记录指向服务器IP
```

**选项B：使用华为云提供的临时域名**
```bash
# 华为云会提供类似：123-456-789-012.compute.xxx.myhuaweicloud.com
```

### 3. 自动化部署脚本

使用我们提供的一键部署脚本，10分钟完成部署。

### 4. 前端配置

只需更新一个环境变量：
```bash
CAMPUS_PDF_SERVICE_URL=https://pdf.你的域名.com/generate-pdf
```

## 成本估算

- **服务器**：约30-50元/月（2核4GB）
- **带宽**：约20-40元/月（5Mbps）
- **域名**：免费或约50元/年
- **SSL证书**：免费（Let's Encrypt）

**总计：约50-90元/月**

1. **完美兼容**：解决所有网络和安全问题
2. **性能优秀**：华为云网络质量好
3. **独立可控**：不依赖校内网络
4. **成本合理**：比阿里云、腾讯云便宜
